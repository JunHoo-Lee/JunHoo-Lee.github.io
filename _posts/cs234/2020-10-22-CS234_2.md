## Computing the Value of a Marcov Reward Process

- Generate a large number number of episodes
- Average returns
- Concentration inequalities bound how quickly average concentrates to expected value
- It doesn't require marcov assumption

$$
V(s) = R(s) + \gamma\sum_{s^`\in S}P(S^` | s) V(s^`)
V = R + \gamma PV
$$

- in this case, we know P and R, so It would be
- $V = {(I-\gamma P)}^{-1} R$
- Equation in MRP

## Iterative Algorithm for Compution Value of a MRP

$$ V_k(s) = R(s) = \gamma \sum_{s^`\in S}P(s^` | s) V_{k-1}(s^`) $$

- Policy specifies what action to take in each state

### MDP+Policy

- MDP + $\pi(a|s)$ = Markov Reward process
  $$ R^{\pi}(s) = \sum_{a\in A}\pi(a|s)R(s,a)
  \\ P^\pi(s^`|s) = \sum_{a\in A}\pi(a|s)P(s^`|s,a)
  $$
- in this way, we can see MDP as MRP
- Iterative Algorithm
  $$ V^{\pi}_k(s) =  r(s,\pi(s)) + \gamma\sum_{s^`\in S}p(s^`|s,\pi (s))V^{\pi}_{k-1}(s^`)
  $$
  
  - this is called Bellman backup
