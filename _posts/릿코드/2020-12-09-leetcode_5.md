그냥 리스트 엘리멘트중 두개 더해서 이 합의 모듈로 60이 0이 되는 조합의 개수를 구하면 되는 문제이다.

이중 특이한 경우는 모듈로 0이 되는 엘리먼트와, 30이 되는 엘리먼트이므로(이 경우는 $n\choose 2$) 이 경우는 따로 빼줘서 처리해주었다.

```python
class Solution:
    def numPairsDivisibleBy60(self, time: List[int]) -> int:
        ret = 0
        modulo = [0] * 60
        for i in time:
            modulo[i%60] +=1
        for i in range(1,30):
            ret += max(modulo[i],0)*max(modulo[60-i],0)
        
        ret += int(max(modulo[0],0)*max(modulo[0]-1,0)/2)
        ret += int(max(modulo[30],0)*max(modulo[30]-1,0)/2)
        
        return ret
```
