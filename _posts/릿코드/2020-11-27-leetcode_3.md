즉 문제상황은 다음과 같다. 1,11,111,1111 .... 로 나가는 수열에서

주어진 K가 이 수열 중 하나를 나눌 수 있느냐? 를 물어보며 나눌 수 있을때, 가장 작은 수열의 길이를 return해라 라는 뜻이다.

아이디어는 간단하다.

$$a_n = 10a_{n-1} + 1$$
이므로,
$$
a_n\%K = 10a_{n-1}\%K + 1\%K
$$

$a_n$을 $10(a_{n-1}\%K) + 1$이라고 보는 것이다

그리고 이 숫자가 계속해서 1씩 달라지므로, 만약 K번 시도했는데도 실패했다면 이것은 영원히 나눠떨어지지 않는다고 간주하여 -1을 리턴. 아니면 계산한 값을 리턴하게 하였다

import re

class Solution(object):
    def calculate(self, s):
        """
        :type s: str
        :rtype: int
        """
        # whitespce를 없애고
        sentence = s.replace(" ","")
        # 연산기호들로 split한 결과를 operand로 저장하고
        operand = re.split("\*|\+|\-|\/|\\s+",sentence)
        b = re.split("\\d+",sentence)
        operator = []
        # 이다음에 whitespace로 분리했더니 이상한 whitespace가 남아서 

        for op in b:
        # For문으로 풀어주고..
            if op == "*" or op == "/" or op == "-" or op == "+":
                operator.append(op)
        newoperand = []
        newoperator = []
        expected_length = 1
        # 첫번째 for 문에서 곱하기를 제거한 후
        if(len(operator)>0):
            for opidx in range(len(operator)):
                op = operator[opidx]
                expected_length += 1
                if op == "*":
                    expected_length -= 1
                    if expected_length <= len(newoperand):
                        num = int(newoperand[-1]) * int(operand[opidx + 1])    
                        newoperand[-1] = num
                    else: 
                        num = int(operand[opidx]) * int(operand[opidx + 1])
                        newoperand.append(num)
                elif op == "/":
                    expected_length -= 1
                    if expected_length <= len(newoperand):
                        num = newoperand[-1] / int(operand[opidx + 1])    
                        newoperand[-1] = num
                    else: 
                        num = int(operand[opidx]) / int(operand[opidx + 1])
                        newoperand.append(num)
                else:
                    newoperator.append(operator[opidx])
                    if opidx > 0 and (operator[opidx - 1] == "*" or operator[opidx -1] == "/"):
                        pass
                    else :
                        newoperand.append(int(operand[opidx]))

                    if opidx == len(operator) -1:
                        newoperand.append(int(operand[opidx+1]))
        else: return int(operand[0])

        # 두번째에서 더하기를 제거한다.
        if len(newoperand) != 0:
            ret = newoperand[0]
            for i in range(len(newoperator)):
                op = newoperator[i]
                if op == "+":
                    ret += newoperand[i+1]
                else:
                    ret-= newoperand[i+1]
        return ret
