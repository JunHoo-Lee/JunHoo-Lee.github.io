주어진 문자열의 K번째 수를 리턴하라.

## Decoded String at Index

![문제](/images/DecodedString.png)

내가 생각한 답보다 실제 답이 훨신 간단해서 놀랬다.

처음엔 단순히 주어진 패턴대로 문자열을 만든 후, K번째 문자를 리턴하는 식으로 생각했는데, timeout이 떴다.

이를 해결하기 위해 주어진 decoding pattern대로 인덱스를 유추해 리턴하는 코드를 짜려 했지만.. 너무 복잡한 코드가 나와버렸다

실제로는 굉장히 간단하게 리턴하는데, 원리는 다음과 같다.

1. 패턴대로 주어진 문자열의 길이를 증가시키다가, 주어진 K보다 사이즈가 커지면 멈추고,
2. 패턴대로 길이를 감소시켜서면서 패턴의 길이가 0이 될때까지 반복하는 것이다.

코드는 이렇다

```python
class Solution(object):
    def decodeAtIndex(self, S, K):
        size = 0
        # Find size = length of decoded string
        for c in S:
            if c.isdigit():
                size *= int(c)
            else:
                size += 1

        for c in reversed(S):
            K %= size
            if K == 0 and c.isalpha():
                return c

            if c.isdigit():
                size /= int(c)
            else:
                size -= 1
```

좀 아쉽다. 저렇게 간단하게도 생각할 법 했는데 너무 어렵게 꼬아서 생각했다.
