단순한 inorder traversal 문제이다.

## Validate Binary Search Tree

![Validate Binary search Tree](/images/2020-12-18-11-53-05.png)

나는 DP로 풀었는데, given node의 subtree가 조건을 만족하고, 그 subtree의 각각 node가 조건을 만족하는 node이면 통과되게 recursive하게 짰다.

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def cbst(self, node: TreeNode,lmin = False, rmax = False) -> bool:
        ltrue = True
        rtrue = True
        nval = node.val
        if node.right:
            rnode = node.right
            rval = rnode.val
            rtrue = (nval < rval) and self.cbst(rnode, nval, rmax)
            if rmax != False:
                rtrue = rtrue and (rval < rmax)
            
        if node.left:
            lnode = node.left
            lval = node.left.val
            ltrue = (nval > lval) and self.cbst(lnode,lmin,nval)
            if lmin != False:
                ltrue = ltrue and (lmin < lval)
        
        if rtrue and ltrue:
            return True
        
        return False
    
    def isValidBST(self, root: TreeNode) -> bool:
        return self.cbst(root)
        
```

근데 풀고나서 생각해보니, 어치피 결국 well-defined BST라면 inorder traversal할 때 sorted array로 나올 것이고, 따라서 inorder traversal하는 궤적이 sorted인지만 체크해주면 된다.

이것의 개념을 조금 일반화하면, 예를들어 k번째로 큰 값을 BST에서 찾거나 하는 문제도 같은 방식으로 풀 수 있다.
