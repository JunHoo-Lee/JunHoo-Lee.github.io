다음과 같은 상황이다. 자료구조 배운지 하도 오래되서 그냥 DP로 풀었는데, 방식은 다음과 같다.

root 노드 기준으로, 최대값은 다음과 같을 것이다.

root.left 노드단을 제외한 곳까지 털수 있는 값을 l1, 노드단 합친 곳까지 쳐서 털수 있는 값을 l2라고 하고, 마찬가지로 다른 쪽의 값을 r1,r2라고 정의하면

1. root node vaue + l1 + r1
2. max(l1,l2) , max(r1,r2)
   
라서, 이를 DP로 구현하였다.

```python
def rettwo(self,node):
        """
        :type root: TreeNode
        :rtype: int
        """
        if node.left != None and node.right != None:
            ylmax , nlmax = self.rettwo(node.left)
            yrmax , nrmax = self.rettwo(node.right)
            below = nlmax + nrmax
            bbelow = ylmax + yrmax
            blbelow = ylmax + nrmax
            brbelow = nlmax + yrmax
            canrob = max(below,bbelow, brbelow, blbelow)
            cantrob = bbelow + node.val
            print("we can rob", canrob, "we can't rob", cantrob)
            return canrob, cantrob
        elif node.left != None:
            ylmax , nlmax = self.rettwo(node.left)
            canrob = max(nlmax, ylmax)
            cantrob = ylmax + node.val
            return canrob, cantrob
        elif node.right != None:
            yrmax , nrmax = self.rettwo(node.right)
            canrob = max(nrmax, yrmax)
            cantrob = yrmax + node.val
            return canrob, cantrob
        else:
            return 0, node.val 
def rob(self, root, iamroot = True):
    """
    :type root: TreeNode
    :rtype: int
    """
    if root == None:
        return 0
    ymax , nmax = self.rettwo(root)
    return max(ymax , nmax)
```

```python
def rob(self, root):
        """
        :type root: TreeNode
        :rtype: int
        """
        def helper(root):
            if not root:
                return 0, 0 
            
            l_prev, l_rob = helper(root.left)
            r_prev, r_rob = helper(root.right)
            
            prev = max([l_prev+r_prev, l_rob+r_rob, l_prev+r_rob, l_rob+r_prev])
            rob = l_prev+r_prev+root.val
            return prev, rob
        return max(helper(root))
```
